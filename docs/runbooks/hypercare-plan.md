# Hypercare Plan (Stage 8)

## Window
- Day 0 to Day 14 after go-live.

## Daily checks
- `/health` availability
- KPI summary response time and success rate
- Daily operations writes count
- Export success rate

## Ownership rotation
- Day shift owner: Product/Operations
- Technical owner: API engineer
- Backup owner: platform engineer

## Exit criteria
- 7 consecutive days without P1 incidents
- UAT backlog closed or deferred with approval
- Stakeholder sign-off documented
